Islam Companion

Provides software tools and services for learning about Islam.

Email subscription service

The Email subscription service allows users to receive Holy Quran and Hadith by email.

Holy Quran and Hadith readers

The Holy Quran Reader allows reading and listening to the Holy Quran in Urdu

The Hadith Reader allows reading Hadith in Urdu language.

Hadith Database

The Hadith Database contains Hadith data. The data was obtained with permission from quranurdu.com and hadithcollection.com

The data may be used by web developers in their own applications. No permission is required to use the data.

Open Source

The source code for the Islam Companion website and Islam Companion Web API are availble free of charge.

The source code may be customized and used with other applications.
